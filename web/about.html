Our mission

Build the best AI tool for construction.

What openmud is

openmud is an open-source, agentic AI platform for heavy civil and underground utility construction. The goal is not a chatbot. The goal is AI that executes real workflows from start to finish.

Build mode

Everything is open source while we are in build mode. We are focused on shipping practical tools that save real time in the field and office, then improving them in public with contractor feedback.

What we are building now

We are building an agentic platform for construction teams to execute and automate work: document understanding, estimating, proposals, schedules, email workflows, and project operations.

Current priority is reliability and execution quality across web and desktop so teams can use openmud daily.

Open source and community

openmud is community-driven. If you want to contribute, review code, or follow progress, use the links in Settings for GitHub and Discord.
