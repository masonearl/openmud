Built-in features

Quick estimate

Material, labor, equipment costs. Pipe (4/6/8 inch), concrete, rebar. LF, soil type, crew size. Confidence ranges based on scope and location.

Proposal generator

Bid documents from estimates. Client, scope, total, line items. Export to PDF. Refine with the AI—no more starting from scratch.

Schedule builder

Gantt-style project schedules. Phases, duration, start date. Phase breakdowns, crew sizing. Export to PDF.

Projects & documents

Create projects, upload docs, organize by job. Chat history and files stay with each project.

Construction Q&A

Pipe cost, labor rates, bedding, soil types, bidding. mud1 knows construction—ask anything.

Automations (desktop app)

The Mac app includes local tools that run on your machine:

  • Clean Desktop – Organize files by type (images, documents, archives). Creates project folders when 3+ files share a name.
  • Clean Downloads – Same for your Downloads folder. Keep bids and specs organized.
  • Quick estimate – Full estimating tools with material, labor, equipment rates.
  • Proposal & schedule – Generate PDFs directly from chat.
