Automations

Custom workflows that connect openmud to your tools.

We’re building automations so you can run workflows like “clean up my desktop” or “find that email from PlanHub.” Here’s what’s possible and what’s coming.

Desktop app

Get openmud as a native Mac app. It’s a simple wrapper around openmud.ai—same experience, always in your dock. Future automations (desktop cleanup, local file access) will run through this app.

Download for Mac

macOS 10.15 or later. Requires Python 3.

Web vs desktop

From the web (openmud.ai): We can connect to services you authorize via OAuth. For example, “find an email in my inbox” works if you connect Gmail or Outlook—you grant permission once, and openmud can search your mail through their APIs. No desktop app required.

From a desktop app: Actions like “clean up my desktop” or “organize my Downloads folder” need direct file-system access. Browsers can’t do that for security reasons. For those workflows, we’ll offer a small desktop agent you install locally. It runs on your machine, you approve what it can touch, and openmud (openmud.ai) talks to it to run those automations.

TL;DR: Email search → web (OAuth). Desktop cleanup → desktop agent. Both are on the roadmap.

Coming soon

  • Email search – Connect Gmail or Outlook, then ask “find the quote from PlanHub” or “emails from last week about the Main St project.”
  • Desktop cleanup – A local agent that organizes your Desktop, Downloads, or project folders. You choose what it can access.
  • Calendar & scheduling – Block time, check availability, sync with construction schedules.
